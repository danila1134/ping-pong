# ping-pong
В репозитории лежит прототип игры 'Ping pong', это мультиплеерная игра!
Игра реализована на Python с помощью библиотеки Pygame.
1 Игрок будет управлять левой ракеткой клавишами WASD, а 2 игрок будет правой ракеткой стрелочками вверх, вниз, право, лево
В игре будет мячик, который летит с постоянной скоростью к игрокам, а они должны его отбить.
Если какой то из игроков пропустит мяч мимо себя, то он проиграл.
